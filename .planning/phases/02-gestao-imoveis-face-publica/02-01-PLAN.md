# Plan 02-01: Gestão de Imóveis (CRUD Admin)

**Goal:** Implementar o gerenciamento completo de imóveis no painel administrativo, permitindo criação, edição e exclusão de propriedades com suporte a rich text para descrições.

---

## 1. Preparation & Setup

- [x] Instalar dependências do Tiptap: `@tiptap/react`, `@tiptap/starter-kit`, `@tiptap/extension-link`.
- [x] Instalar `isomorphic-dompurify` (já está no package.json, mas verificar se está pronto para uso).
- [x] Adicionar item "Imóveis" em `src/components/admin/nav-items.ts`.

---

## 2. Shared Components & Schema Refinement

- [x] Criar Zod schema para validação de imóveis em `src/lib/validations/property.ts`.
- [x] Criar componente `RichTextEditor` em `src/components/admin/RichTextEditor.tsx` usando Tiptap.

---

## 3. Server Actions

- [x] Criar `src/actions/properties.ts`:
    - `createProperty(data)`: Sanitiza HTML com DOMPurify, valida com Zod, gera slug, salva no banco e audita.
    - `updateProperty(id, data)`: Mesma lógica para edição.
    - `deleteProperty(id)`: Soft-delete ou exclusão física com auditoria.
    - `togglePropertyStatus(id, status)`: Atalho para ativar/desativar rápido.

---

## 4. UI — Admin Pages

- [x] **Property List** (`src/app/(admin)/admin/imoveis/page.tsx`):
    - Tabela com nome, slug, status (Badge), capacidade e data de criação.
    - Botão "Novo Imóvel".
    - Ações: Editar, Ver na Web, Excluir.
- [x] **Property Form** (`src/app/(admin)/admin/imoveis/new/page.tsx` & `[id]/edit/page.tsx`):
    - Formulário multi-seção: Info Básica, Localização, Configurações (limpeza, noites mínimas), Descrição (RichText).
    - Exclusão de upload de imagens para o próximo plano (02-02).

---

## 5. Security & Validation

- [x] Garantir que `description` seja sanitizada antes de persistir no DB.
- [x] Verificar permissão de `owner` ou `staff` (edit) em todas as actions.

---

## Success Criteria

1. Admin acessa `/admin/properties`, vê a lista de imóveis.
2. Admin cria um novo imóvel com descrição formatada (bold, links).
3. O HTML salvo no banco não contém scripts maliciosos (auditado manualmente).
4. O redirecionamento após salvar funciona e exibe mensagem de sucesso (Sonner).
